# java-chicken-2019
## 프로그램 소개
- 치킨집 사장님이 사용하는 간단한 포스(POS)프로그램을 구현한다.
- 주문등록, 결제하기, 프로그램종료 기능을 가진다.
- 메뉴 기본 정보가 주어지며 메뉴번호, 종류, 이름, 가격을 가진다.
- 테이블 기본정보가 주어지며 테이블 번호를 가진다.
- 한 테이블에서 주문할 수 있는 한 메뉴의 최대 수량은 99개이다.
- 주문이 등록된 테이블은 결제가 이루어지기 전까지 테이블 목록에 별도로 표시한다.
- 주문 내역에 대한 계산을 할 때는 결제 유형에 따라 할인율이 달라진다.
- 치킨 종류 메뉴의 수량 합이 10개가 넘는 경우 10,000원씩 할인된다.
- 현금 결제는 5%가 할인되며 할인된 금액에서 한 번 더 할인이 가능하다.
- 주문 혹은 결제가 불가능한 경우 그 이유를 보여주고, 다시 주문 혹은 결제가 가능하도록 해야 한다.
- 최종 결제 금액을 보여준다.

## 구현할 기능 목록
1. 메인화면 구현
    * 주문등록, 결제하기, 프로그램 종료를 출력하고 선택하도록 한다.
    * [예외처리] 숫자가 아닌 경우
    * [예외처리] 주문등록, 결제하기, 프로그램 종료에 해당하는 숫자 1,2,3이 아닐 경우
    
1. 주문등록 - 테이블 선택
    * 테이블 목록을 보여준다.
    * 테이블을 선택한다.
    * [예외처리] 최대한도(99)까지 이미 주문한 경우
    * [예외처리] 숫자가 아닌 경우
    * [예외처리] 테이블 목록에 있는 숫자가 아닌 경우 
    
1. 주문등록 - 메뉴 선택
    * 메뉴 목록을 보여준다.
    * 메뉴와 수량을 선택한다.
    * [예외처리] 숫자가 아닌 경우
    * [예외처리] 테이블 목록에 있는 숫자가 아닌 경우 
    * [예외처리] 수량이 0 이하일 경우
    * [예외처리] 주문량이 99를 초과할 경우
    
1. 주문등록 - 정보 저장
	* 주문 정보를 저장한다. 	
	* 주문이 완료되면 메인화면으로 돌아간다.

1. 결제하기 - 테이블 선택
    * 테이블 목록을 보여준다.
    * 주문이 있는 테이블은 (W)을 붙여서 보여준다.
    * 테이블을 선택한다.
    * [예외처리] 숫자가 아닌 경우
    * [예외처리] 테이블 목록에 있는 숫자가 아닌 경우 
    * [예외처리] 주문이 없을 경우

1. 결제하기 - 주문 내역 출력
    * 주문 내역을 보여준다.

1. 결제하기 - 결제 방법을 선택한다.
    * 신용 카드, 현금을 선택한다.
    * [예외처리] 숫자가 아닌 경우
    * [예외처리] 신용 카드, 현금에 해당하는 숫자 1,2가 아닐 경우
    
1. 결제하기 - 결제 진행
    * 치킨 종류 메뉴 수량 합이 10개가 넘는 경우 10000원씩 할인된다.
    * 현금 결제는 할인된 금액에서 5% 추가 할인된다. 
    * 결제가 불가능 할 경우 이유를 보여주고, 다시 결제를 하도록 한다.
    * 결제가 가능할 경우 최종 결제할 금액을 출력한다.
    * 결제가 끝나면 해당 테이블에 있는 주문 정보를 초기화한다.
    * 결제가 완료되면 메인화면으로 돌아간다.

1. 프로그램 종료
    * 프로그램을 완전히 종료한다.
